Admin Quickstart Guide

How to log in and out of your admin account on the SWPPP web portal.

1 Open the Login Page

Navigate to the login page in your browser:

Production: https://sw3p.pro/auth/login
Local dev: http://localhost:8001/auth/login

If you're already logged in, the page will automatically redirect you to the portal.

2 Sign In with Your Password

The login page has two tabs: Access Code and Password.

  1. Click the Password tab.
  2. Enter your Name (the display name you chose when you first claimed your invite).
  3. Enter your Password.
  4. Click Sign In.
Tip: If you haven't set a password yet, use your original access code on the Access Code tab to log in first, then set a password from the portal settings (gear icon).
Access codes are one-time use. Once you've claimed a code, it cannot be reused. Use password login for all future sign-ins.

3 You're on the Portal

After signing in, you land on the Company Tools portal. From here you can:

  1. Click any app card to open that tool (e.g. SWPPP AutoFill).
  2. Click Admin (top-right, visible only to admins) to open the Admin Panel.
  3. Click the gear icon to set or change your password.

4 Log Out

To end your session:

  1. From any portal page, look at the top-right corner of the header.
  2. Click the Logout button.
  3. You'll be redirected back to the login page. Your session cookie is cleared.

Quick Reference

Login URL https://sw3p.pro/auth/login
Portal URL https://sw3p.pro/
Admin Panel https://sw3p.pro/admin
Sign-in method Name + Password (preferred) or one-time Access Code
Log out Click Logout in the top-right header
